# AgriVision-DeepLabV3
## Plant Disease Joint Diagnostics.
This project tackles the limitations of current agricultural disease detection models by introducing a joint classification-and-segmentation deep learning pipeline.

Built on DeepLabV3+ with a ResNet-101 backbone and optimized via a custom multi-task loss function, the system moves beyond simple image-level labels to provide precise, pixel-level mapping of infected tissues.

Trained on comprehensive datasets like PlantVillage and PlantDoc, it successfully generalizes to real-world field environments across multiple crop types, ultimately improving classification accuracy by 2.3% and providing actionable severity metrics for precision agriculture.
